## How To Install Quarrel?

Its Simple!

For **Powercord** Users:

1. Open CMD & Type:

```
cd powercord/src/Powercord/themes && git clone https://github.com/leeprky/Quarrel
```

For **Betterdiscord** Users:

1. CLick > [Download](https://betterdiscord.app/Download?id=246)

2. Download Raw
